## 👋 關於我們

**中國移動（香港）創新研究院有限公司** 是面向國際科技創新合作而設立的創新研究平台，致力連接全球研發資源，強化跨境協同創新，支撐中國移動核心能力與產品走向國際市場。

我們立足香港、面向世界，發揮香港作為國際橋樑的獨特優勢，構建全球協同的研發創新生態。我們重點推進高質量國際科技合作、全球創新人才匯聚、產學研用協同，以及國際組織營運支撐。

我們的使命是助力香港建設國際創新科技中心，同時推動全球資訊通信產業的數碼化、智能化轉型。

---

## 🚀 戰略重點方向

### 人工智能

我們聚焦大模型應用落地、大模型前沿研究探索，以及面向真實產業場景的 AI 解決方案整合與應用。

### Web 3.0

我們開展 Web 3.0 技術、應用及新興產業生態研究，重點關注可信數碼基礎設施和下一代數碼服務。

### 6G

我們推進 6G 前沿技術研究、國際標準制定、智能網絡融合，以及面向未來流動資訊系統的場景驗證。

---

## 🌐 我們的工作

### 服務能力「出海」

我們支持國內產品與技術能力面向國際市場進行適配與創新，重點服務「一帶一路」等全球市場，助力核心能力走向海外。

### 國際科研合作與引才

我們與高校、科研機構、企業及產業夥伴合作，圍繞 AI、Web 3.0、6G 等前沿領域推動關鍵技術突破，並吸引全球頂尖科研人才參與創新合作。

### 支撐國際組織營運

我們支持包括 GTI 相關工作在內的國際產業合作平台營運，強化亞太、「一帶一路」沿線及全球資訊通信產業生態合作。

## 🧭 長期願景

我們致力於在 2026–2030 年期間，發展成為領先的國際化新型研發機構。

我們的發展路徑包括：

* **2024 年：** 在香港實現起步，並逐步融入全球創新網絡。
* **2025 年：** 建章立制、規範發展，逐步形成國際科創影響力。
* **2026–2030 年：** 成為具備全球協同能力和持續創新影響力的領先國際化研發機構。
